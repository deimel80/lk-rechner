# WTV / DTB LK-Rechner PWA

Eine kleine, eigenständige Web-App für iPhone/Android/Desktop.

## Dateien

- `index.html` – Oberfläche
- `style.css` – Design
- `app.js` – LK-Berechnung
- `manifest.json` – App-Installation
- `sw.js` – Offline-Cache
- `icons/` – App-Icons

## GitHub Pages veröffentlichen

1. Bei GitHub ein neues Repository erstellen, z. B. `lk-rechner`.
2. Alle Dateien aus diesem ZIP in das Repository hochladen.
3. In GitHub auf **Settings → Pages** gehen.
4. Unter **Build and deployment** als Source **Deploy from a branch** wählen.
5. Branch `main` und Ordner `/root` auswählen.
6. Speichern.
7. Nach kurzer Zeit ist die App erreichbar unter:
   `https://DEIN-GITHUB-NAME.github.io/lk-rechner/`

## Auf iPhone installieren

1. Link in **Safari** öffnen.
2. Teilen-Symbol antippen.
3. **Zum Home-Bildschirm** wählen.
4. Namen bestätigen, z. B. `LK-Rechner`.
5. Die App liegt danach wie eine normale App auf dem Home-Bildschirm.

## Hinweis

Der Rechner bildet die LK-Formel lokal nach. Offiziell maßgeblich ist die zentrale DTB-/NTDB-Auswertung.
